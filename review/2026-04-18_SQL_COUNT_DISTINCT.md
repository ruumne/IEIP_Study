# 📝 2026年 第1回 情報処理技術者試験 実技 - 誤答ノート (SQL)

## 📌 問題の概要 (9番)
`STUDENT` テーブルに以下の学生データが格納されている場合、3つの SQL クエリの実行結果（行数/タプル数）を予測しなさい。

**[テーブルのデータ分布]**
- コンピュータ学科: 50名
- 電算科: 100名
- 電気科: 50名
- 合計タプル数: 200名

### クエリ 1
```sql
SELECT DEPT FROM STUDENT;
```

### クエリ 2
```sql
SELECT DISTINCT DEPT FROM STUDENT;
```

### クエリ 3
```sql
SELECT COUNT(DISTINCT DEPT) FROM STUDENT WHERE DEPT = '電算科';
```

---

## ❌ 私の解答 (My Answer)
`3, 200, 100`

## ✅ 正解 (Correct Answer)
`200, 3, 1`

---

## 🔍 なぜ間違えたのか？ (原因分析)

SQLの基本である `DISTINCT` と `COUNT` の組み合わせに対する理解不足が原因でした。

### 1. `SELECT DEPT FROM STUDENT;` (正解: 200)
- `WHERE` 条件や `DISTINCT` 指定がないため、テーブルに存在するすべてのタプル（200名分）の学科情報がそのまま出力されます。
- **誤解した点:** 重複が排除されて学科の種類（3）が出力されると勘違いしました。

### 2. `SELECT DISTINCT DEPT FROM STUDENT;` (正解: 3)
- `DISTINCT` キーワードにより、重複する学科名が排除されます。
- 存在する学科は「コンピュータ学科」「電算科」「電気科」の3種類だけなので、結果は3行になります。
- **誤解した点:** 単純な `SELECT` 文の結果である全タプル数（200）と混同してしまいました。

### 3. `SELECT COUNT(DISTINCT DEPT) FROM STUDENT WHERE DEPT = '電算科';` (正解: 1)
- まず `WHERE DEPT = '電算科'` で、電算科の学生100名のデータに絞り込まれます。
- その結果に対して `DISTINCT DEPT` を適用すると、「電算科」という1種類の値だけが残ります。
- 最後に `COUNT` でその種類数を数えるため、結果は `1` となります。
- **誤解した点:** `COUNT` を見て、条件に合致する「学生の人数（100名）」を数えるクエリだと早とちりしてしまいました。
