IEIP_Study (情報処理技術者試験 実技対策)
Information Processing Engineer Practical Exam Study Log

📌 概要 (Overview)
韓国の国家資格である「情報処理技術者試験 (実技)」の合格に向けた学習記録と、アルゴリズム問題のトレーシングノートです。
(This repository contains study notes and algorithm tracing logs for the Korean national certification, 'Information Processing Engineer (Practical)'.)

🎯 目標 (Goal)
2026年 第2回 実技試験 合格 (2026年 7月~8月 予定)

C, Java, Python コードの正確な出力結果予測 (手書きトレーシング)

SQL クエリ作成およびデータベース設計の理解

IT関連の基礎理論 (要求事項分析、デザインパターン、ネットワークなど) の暗記と定着

📝 記録内容 (Contents)
1. アルゴリズム トレーシング (Algorithm Tracing)

 過去問のコード (C, Java, Python) を手作業でトレースし、変数の変化と出力結果を記録します。

2. SQL & DB

SELECT, JOIN, サブクエリなどの基本的なクエリ作成練習。

3. 誤答ノート (Review Notes)

模擬試験や過去問で間違えた問題 (例: ネットワークのCIDR表記漏れ、要求事項の順序ミスなど) の原因分析と復習。
